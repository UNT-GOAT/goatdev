Prod Integration Example

Reference implementation — camera streams, grade submission, status polling, debug images.
All requests go to the Pi. The Pi captures images and forwards to EC2 internally.

Backend services on the Pi used by this page:
• Camera proxy (port 8080) — MJPEG streams at /api/viewfocus/stream/{cam}
• Prod server (port 5000) — /api/prod/grade, /api/prod/status, /api/prod/debug/{id}/{view}
• Caddy reverse proxy unifies both behind port 8443. It strips the /api/prod prefix before forwarding to port 5000.
• See the included Caddyfile under pi/system/ for the full routing config.
